## BaseActivity Usage
> * getPresenter返回对应P,与当前BaseUI绑定
> * getDefaultLayout 返回加入的布局文件
> * onViewAdded 初始化View操作
> * startLoading/stopLoading/onEmpty是可用被调用的方法,展示不同状态视图层
